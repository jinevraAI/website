Why is jinevra special?

Jinevra democratizes access to powerful natural language processing tools to fill gaps in existing tools and to tackle some persistent challenges in language classification and bias detection in English.

Python Package

In the first half of 2019, install our package to use on your coding projects.

Jupyter Notebooks

If you know just a little Python and how to run a web server, the Jupyter Notebooks will be a walkthrough of the functionality in the Python package.

No code? No problem!

Coming in the second half of 2019, anyone will be able to run a text analysis on this website with no code for a small fee.

Open Source

Python Packages and the Jupyter Notebooks are open sourced under an MIT license.

Why this, why now?

Jinevra is an open source first project with a mission to give everyone access to AI-based language analysis tools.

This project is being built to provide easy access to sophisticated textual analysis tools for journalists, writers, and academics in the humanities and social sciences. Python programmers can also use the package to complement well-known libraries for Natural Language Processing.

Jinevra is led by Eleanor Stribling. If you'd like to join the team to collaborate on development, documentation or outreach, please send an email to hello@jinevra.ai or subscribe to the mailing list.
